# Визуализация графов. Интерактивная  визуализация  в Plotly и Pyvis.
## Что из себя представляет проект
Проект выполнен в рамках практического задания №4 по визуализации графов.

Цель — анализ финансовых транзакций в системе мобильных денежных переводов (mMoney) для обнаружения мошеннических схем с использованием библиотек Plotly и Pyvis.

Исследуются два типа мошенничества:
- Мобильный ботнет (заражение устройств) — множество заражённых пользователей переводят деньги «мулам», которые затем обналичивают средства через агентов.
- Кража мобильного телефона — многократные попытки снятия небольших сумм в короткий промежуток времени.
## Данные
- Размеченный набор данных — используется для анализа и понимания признаков мошенничества.
- Неразмеченный набор данных — основной объект исследования; необходимо определить тип мошенничества и вовлечённых пользователей.

Транзакции содержат поля: 
- ID отправителя/получателя, 
- роли (EU, Mer, RET), 
- сумма, 
- тип транзакции (Ind, Dt, Wl, Merchant), 
- статус, 
- временные метки, 
- балансы.
## Результаты
- В неразмеченном наборе данных обнаружены оба типа мошенничества.
- Для ботнета определены 4 пользователя-«мула» (PN_EU_0_955, PN_EU_0_260, PN_EU_1_328, PN_0_1045) и множество заражённых узлов (45 пользователей).
- Для кражи телефона выявлено 137 подозрительных аккаунтов, совершавших множественные снятия средств через разных агентов в течение короткого времени.
Иллюстративные материалы (проекции PCA, графы контактов) подтверждают структуру мошеннических схем.
